On initial download, run these commands at the top directory:

    npm install react react-dom
    npm install webpack webpack-dev-server babel-loader babel-preset-es2015 babel-preset-react babel-preset-stage-2
    npm install babel-core
    npm install prop-types
    npm install react-router
    npm install react-router-dom
    npm install css-loader
    npm install style-loader

To run the build version of this prototype, execute:
    npm start

Project will be running on localhost:8080
